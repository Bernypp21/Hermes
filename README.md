# Hermes
Chat app made for CS314 Element software engineer at PSU.  I fully developed the backend server and components for the app and received the frontend part from the TA.



The backend use a MERM frame work
    - MongoDB: A database for storing and retrieving data
    - Express: a web application for build APIs
    - React: A JavaScript library for building user interfaces/
    - Node.js: A runtime for executing Javascript


functionality I created for the backend server:

    *User authentication(registration,login,logout)
    * data handling in a MongoDB database
    * maintain database
        - use a hashing function to store user's password hash
    * allowed for real time messaging between connected  clients

    *database management
        - store and retrieve data efficiently
        - handle error cases
